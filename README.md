# ⚡ Energy Consumption Calculator

A Java-based application that helps users calculate and analyze their household energy consumption, cost, and carbon footprint.

## 🚀 Features

* Add and manage home appliances
* Calculate daily and monthly energy usage
* Estimate electricity cost
* Calculate carbon footprint (CO2 emissions)
* Identify highest energy-consuming appliance
* Provide smart energy-saving tips
* Renewable energy analysis (solar & wind)
* Save monthly and annual consumption data

## 🛠️ Technologies Used

* Java (OOP concepts)
* Java Swing (GUI)
* File Handling
* Collections (ArrayList, HashMap)

## 📊 Core Concepts

* Abstraction (Appliance class)
* Inheritance & Polymorphism
* Interfaces (CarbonCalculator)
* Encapsulation
* Separation of concerns (Model / Service / Util)

## ▶️ How to Run

1. Open the project in IntelliJ or Eclipse
2. Run `Main.java`
3. Start adding appliances and analyzing energy usage
